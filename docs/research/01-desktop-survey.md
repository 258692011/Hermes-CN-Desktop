### 第 4 部分 · 桌面端功能优先级（3 题，核心）

#### Q10 · 假设桌面版要分阶段做，你最希望**前 5 个先做的功能**是？

- **题型**：多选 · 必填 · **恰好 5 项**（不可多不可少）
- **选项池（15 项，乱序展示）**：
  - 多模型一键切换（OpenAI / Anthropic / 国产 LLM）
  - 拖拽文件 / 文件夹直接当任务附件
  - 任务跑完桌面通知 + 系统托盘进度
  - 多 Profile / 多账号切换（工作 / 个人 / 不同 API key）
  - 全局 ⌘K 命令面板搜会话 / 文件 / 命令
  - 自定义 Skill 可视化编辑器（不用改 YAML）
  - MCP 服务器一键添加 / 在线状态可视化
  - 定时任务（Cron）+ 历史 + 一键重跑
  - 项目内 Git 状态 + 改动文件直观查看
  - 全局快捷键（OS 级，按一下唤起 Hermes）
  - 自动更新（一键升级，不破坏当前任务）
  - 工作区原生集成（点击文件直接 VSCode 打开）
  - 命令行任务模板 / 配方库
  - 跨设备同步（会话 / Skills 云端备份）
  - 协作 / 分享某个会话给同事
- **PM 备注 (D-1 D-8)**：用 Borda count 或 Approval Voting 加权，前 5 票数差 < 10% 视为并列。

#### Q11 · 下面这些功能，**没有就不用** vs **有最好** vs **可有可无**？（矩阵）

- **题型**：矩阵单选 · 必填
- **行 (8 项，跟 Q10 不重叠的)**：
  - 桌面端可以离线启动（不依赖 Dashboard 在线）
  - 数据完全本地（不经云端）
  - 国产 LLM 接入（通义 / 智谱 / DeepSeek / Kimi / 豆包 等）
  - 支持深色 / 浅色主题切换
  - 中文界面（不强制英文）
  - 任务跑完邮件通知
  - 与浏览器集成（hermes:// 链接）
  - 类似 Raycast 的 OS 级命令栏
- **列**：
  - 没有就不用
  - 有最好
  - 可有可无
- **PM 备注**：「没有就不用」票数前 3 = v2 阶段 1 必备。
